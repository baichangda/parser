# 概述
## parser-base
解析框架基础、提供一套注解用于标定bean、完成解析
通过javassist字节码增强生成解析类

## parser-protocol-xxx
解析框架协议实现
通过**parser-base**实现的各类协议解析

# 使用方式
如下地址为依赖包仓库地址
https://central.sonatype.com/namespace/io.github.baichangda
